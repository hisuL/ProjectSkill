## 4. Layout And Regions / 布局与区域划分
### 4.1 Page Skeleton / 页面骨架
```text
┌──────────────────────────────────────────────────────────┐
│  🧠 ChatFile      当前文档信息 / 暂未加载文档   [主题] [❓] │
├───────────────────────────┬──────────────────────────────┤
│                           │                              │
│     💬 对话区（40%）        │     📖 PDF 预览区（60%）      │
│                           │                              │
├───────────────────────────┴──────────────────────────────┤
│ [📎 上传/更换文档] [输入框........................] [➤/■] │
└──────────────────────────────────────────────────────────┘
```

### 4.2 Region Responsibilities / 区域职责
- 顶部导航：
  - 展示产品名、当前文档名、页数、文件大小、主题切换和帮助入口。
- 左栏对话区：
  - 承载欢迎态、消息流和反馈相关交互。
- 右栏 PDF 预览区：
  - 承载 PDF 预览、目录侧栏、缩放、页码导航、高亮和加载失败重试。
- 底部固定输入区：
  - 承载上传/更换文档、问题输入、发送/停止操作。
- 全局浮层：
  - 承载取消上传确认、更换文档确认、快捷键帮助和 toast。

### 4.3 First-screen And Variant Layouts / 首屏与变体布局
- 空态：
```text
┌─────────────────────────────────────────┐
│                                         │
│        📄                               │
│    拖拽 PDF 文件到此处                    │
│    或 点击选择文件                        │
│    支持 PDF，单文件 <= 100MB             │
│         [ 选择文件 ]                     │
└─────────────────────────────────────────┘
```
- 上传中：
```text
┌─────────────────────────────────────────┐
│  📄 XYZ_Datasheet_v2.0.pdf              │
│                                         │
│  ████████████░░░░░░░░  65%  上传中...    │
│                                         │
│             [ 取消上传 ]                 │
└─────────────────────────────────────────┘
```
- 处理中：
```text
┌─────────────────────────────────────────┐
│  📄 XYZ_Datasheet_v2.0.pdf              │
│                                         │
│  ✅ 上传完成  →  🔄 文档解析中...         │
│               →  ○ 智能切片              │
│               →  ○ 向量索引              │
│                                         │
│  预计耗时：约 2 分钟                      │
│                                         │
│        [ 取消处理 ]                      │
└─────────────────────────────────────────┘
```
- ready 态：
  - 步骤条收起为顶部 ready 横幅 `3 秒自动淡出`，页面切换为工作台分栏。
- 失败态：
```text
┌─────────────────────────────────────────┐
│  ❌ 文档解析失败                         │
│                                         │
│  原因：文件已加密，无法提取文本内容        │
│                                         │
│  [ 重新上传 ]    [ 查看详情 ]            │
└─────────────────────────────────────────┘
```
- 折叠预览态：
  - 右侧收为 `40px` 窄条，仅保留展开按钮。
- 更换文档弹层：
  - 点击底部 `[📎]` 后弹出操作菜单，选择“上传新文档”需要二次确认“更换文档将清空当前对话，确定？”。
- 预览区加载变体：
  - 文档未上传时显示灰色占位文案。
  - 文档加载中显示骨架屏和顶部加载进度条。
  - 加载失败显示居中错误文案和 `[重试]`。

## 5. User Flow / 用户流程
- 主流程：
  - 进入空态 -> 上传 PDF -> 显示上传进度 -> 进入处理步骤条 -> ready 横幅 -> 工作台问答。
- 分支流程：
  - 上传中取消 -> 回空态。
  - 处理中取消 -> 回空态并清空中间状态。
  - 点击引用 -> 预览区自动跳页并高亮。
  - 点击 `[📎]` -> 弹出更换文档菜单 -> 确认后清空当前对话并回上传流程。
  - 点击 `[☰ 目录]` -> 目录侧栏滑入 -> 点击章节跳转 -> 目录自动收起。
- 失败回路：
  - 文件校验失败 toast。
  - 解析失败展示错误卡。
  - 预览加载失败仅影响右栏，不阻塞对话。

## 6. API Interaction / API 交互
| Stage / 阶段 | API | Trigger / 触发条件 | Failure Handling / 失败处理 |
| --- | --- | --- | --- |
| 上传 | `POST /api/documents` | 选择文件或拖拽放下 | 失败 toast 或失败卡 |
| 查询状态 | `GET /api/documents/{docId}` | 上传后轮询 | 失败时保留当前 UI 并提示重试 |
| 取消处理 | `POST /api/documents/{docId}:cancel` | 点击取消上传/取消处理 | 失败时保留当前阶段并提示 |
| 预览元数据 | `GET /api/documents/{docId}/preview` | 上传成功后并行加载预览 | 失败时右栏显示预览错误 |
| 对话流 | `POST /api/conversations/query:stream` | 输入区发送 | 错误消息卡 |
| 欢迎态 | `GET /api/conversations/{docId}/welcome` | 文档 ready 且会话为空 | 左栏显示欢迎态骨架或内容 |

## 7. Interaction Design / 交互设计
| Scenario / 场景 | Trigger / 触发条件 | State Change / 状态变化 | Feedback / Motion / 反馈与动效 | Copy / Timing / 文案与时序 | Error / Edge Handling / 异常与边界处理 |
| --- | --- | --- | --- | --- | --- |
| 顶栏信息展示 | 页面加载或文档切换 | 中间区切换为“暂未加载文档”或“文件名 + 页数 + 大小” | 无强动效 | 文档未上传时显示占位文案 | 帮助入口始终可用 |
| 拖拽上传 | 文件拖入页面 | 上传卡进入高亮态 | 蓝色虚线 + 呼吸动效，必要时全屏遮罩淡入 | 松手后回正常态或开始上传 | 非 PDF 直接 toast |
| 点击上传 | 点击上传卡或按钮 | 打开系统文件选择器 | 无额外动效 | 选中文件即进入上传态 | 取消选择不变更页面 |
| 上传中 | 文件提交成功 | 显示百分比和取消按钮 | 线性进度条 | 实时显示 `xx% 上传中...` | 取消需二次确认 |
| 处理中 | 上传完成 | 展示四阶段步骤条，右栏可并行预览 | 当前步骤 loading，已完成打勾，未开始灰圆点 | 预计耗时约 2 分钟 | 允许取消处理 |
| ready 横幅 | 状态变 ready | 步骤条收起，切到工作台 | 横幅淡出 | 3 秒后自动消失，输入框聚焦 |  |
| 更换文档 | 点击 `[📎]` 并选择上传新文档 | 弹出确认框 | 浮层出现/关闭 | 确认后清空当前对话并回上传流程 | 点外部区域关闭菜单 |
| 引用跳转 | 点击消息引用 | 预览区滚动、高亮、必要时先展开 | 平滑滚动约 400ms，高亮 3 次呼吸，每次 600ms | 高亮 3 秒后转浅黄常驻 | 新引用清掉旧高亮 |
| 目录侧栏 | 点击 `[☰ 目录]` | 目录面板覆盖预览区 | 左滑入，遮罩半透明，当前章节高亮 | 选中目录项后自动收起 | 多级目录支持折叠展开 |
| 预览区状态切换 | 文档未上传/加载中/成功/失败 | 右栏在占位、骨架屏、内容、错误卡间切换 | 骨架扫光，顶部加载进度条 | 加载失败时显示 `[重试]` | 不阻塞左栏对话 |
| 缩放与翻页 | 点击按钮、滚轮或输入页码 | 更新当前页和缩放等级 | 缩放平滑、页码即时回显 | 缩放范围 50%-300%，默认 Fit Width | 非法页码回退到最近合法值 |
| 输入框状态机 | 文档状态或消息流变化 | 在禁用/空闲/可发送/生成中切换 | 生成中按钮变红色 stop | 完成后清空并聚焦；占位符随文档状态变化 | 超 2000 字不可发送 |
| 快捷键 | 键盘输入 | 执行发送、换行、停止、聚焦、上传、帮助 | 无额外动效 | `Enter` 发送，`Shift+Enter` 换行，`Esc` 停止或关闭面板 | 生成中禁止重复发送 |
| 服务异常 | `503/timeout/network` | 显示错误消息卡或预览错误卡 | 重试按钮可见 | 3 次连续失败禁用输入 30 秒 | 保留请求 ID |

## 9. Telemetry And Guardrails / 埋点与保护措施
- 埋点：
  - 上传开始、上传完成、ready、引用点击、预览折叠、目录打开、缩放、翻页、发送、stop、重试、更换文档确认。
- 保护措施：
  - 纯空格发送前端拦截。
  - 300ms 防抖避免快速连击发送。
  - 粘贴超长文本自动截断并提示。
  - 非法页码输入回退到合法区间。
